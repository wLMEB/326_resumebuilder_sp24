Overview

Resume_builder is a web application that targets a faster and more efficient way of resume building. Our project aims to help college students and recent graduates have a smoother experience through their job application phase. With Resume_builder, users can input all of their past experiences by categories, including skills, work experience, project experience, etc., into our experience bank. All information entered will be stored in a database by category at the back end so that there will be no need to re-enter the same information twice. There are also options to edit information that has already been input into the experience bank in the case of updating information. When it's time to generate a new resume for a role or company, users will be able to select the skills and experiences they wish to include in their resume, select a template format, and click a generation button. A copy of the resume will be properly formatted and ready to download.

Application Parts

Data Requirements

For our Resume Builder to function as intended, we need to consider certain data requirements that need to be incorporated in order to capture and manage user input. Personal information such as name, contact information, residential address, LinkedIn, and personal website links need to be collected in order to format the desired resume. Following this data, users need to be able to enter their education and work experience. These fields include the name of the user’s educational institution and company, dates of employment and duration of education, and location. Specific to education, the user’s degree, major/field of study, and graduation date are requirements and for work experience, the job title and job description/responsibilities should be incorporated. Additional information that should be collected are projects created and skills acquired from work and education experiences, certifications and achievements, relevant coursework, and proficiency in languages. All of this data will be stored in databases based on categories and user’s will then be able to select specific pieces of information based on what type of resume they need to create for specific roles. Users will be able to change and add additional information if required and the data will be stored in the designated databases based on categories.

Wire Frames

To make our Resume Builder an easy experience for our users, it’s essential that we have components that make it easier for users to visualize the flow of the app and that they understand how to use it. To enable this, our app will include specific pages that will allow users to add their information in steps. The first page will be a landing page that gives users an introduction and some information about how to use the app and options to create a new resume or access a pre-existing one. This page will have labeled buttons that will inform users how to make a new resume or choose an already created one. Once the landing page is accessed, a new page will be displayed where users can input their personal information, work and education information, projects and experiences, and skills. Following this page, a preview page will be displayed where users can see how their final resume will look. There will also be options where users can make any changes or updates to their resume before generating the finalized document. The final page will show the generated resume with options to print or download the document to the local computer.

Real-World Connection

Resume builder: A web application that helps users quickly generate multiple resumes that meet different requirements, providing a variety of templates for users to choose from, helping users easily and conveniently generate different resumes for use in different scenarios. Job seekers often need to submit resumes to multiple companies, and different companies have different requirements for skills and experience, and the things they value are different. There is a lot of time-wasting repetitive work in the process of creating multiple resumes with different needs, such as copying information and reformatting. Our app hopes to help job seekers reduce this repetitive work and create multiple resumes with ease. At the same time, our templates make it easy for first-time resume makers to create a good-looking resume. Especially for users who are not good at typesetting, it can make them avoid the work they are not good at. Users only need to enter their various aspects of data with classification into the application, select specific templates and required data categories to easily generate resumes. Users can easily add, decrease, and change their own data for the data they have already entered. After the user completes the resume design, they can download the designed resume to the local computer.
